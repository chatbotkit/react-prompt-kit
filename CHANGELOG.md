# Changelog

All notable changes to this project will be documented in this file.

## [0.0.2] - 2025-11-07

- Overhauled spacing logic to treat custom XML tags as block-level elements and ensure consistent blank lines in rendered prompts.
- Added whitespace-aware conversion helpers with extensive unit test coverage.
- Moved the demo script into `examples/` and documented how to run it.
- Split TypeScript configuration to keep development and build outputs separate, preventing examples from being bundled.
- Expanded README with rationale for the library and clarified demo instructions.

## [0.0.1] - 2025-11-07

- Initial placeholder release.
